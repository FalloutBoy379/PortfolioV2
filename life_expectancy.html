Life Expectancy Prediction using Machine Learning

Overview

This project leverages multiple machine learning techniques to predict life expectancy from global health indicators. Using a publicly available dataset from the WHO, we explored feature importance, regression modeling, and regularization techniques to improve generalizability.

Dataset & Preprocessing

The dataset included over 3,000 entries across countries and years, featuring predictors like GDP, schooling, BMI, immunization rates, and alcohol consumption. We handled missing data via mean/mode imputation and performed standardization to normalize feature scales.

Approach

  • Linear Regression (Baseline)
  • Ridge and Lasso Regression (for feature selection and regularization)
  • Principal Component Analysis (PCA) for dimensionality reduction
  • Cross-validation for hyperparameter tuning

Results

Our final model used Ridge Regression with tuned alpha = 2.5, achieving an R² of 0.87 on the test set. The most impactful features included schooling, BMI, HIV/AIDS rate, and income composition.

Conclusion

The project demonstrates the power of regularized regression in dealing with multicollinearity and identifying impactful predictors. With minimal preprocessing and tuning, our pipeline provides interpretable results while achieving high accuracy.
